# Microsoft.SqlTools.Sts2.Multiplexer

**Role:** Stdio multiplexer: Content-Length framing, v2 routing, outbound server-request id rewriting, lifecycle mirroring, single stdout writer, crash containment.

**Allowed dependencies:** BCL only (incl. System.IO.Pipelines)

**Forbidden:** STS2 domain logic, legacy namespaces, any package

See docs/sts2/SPEC.md SS4 for the authoritative dependency matrix.
